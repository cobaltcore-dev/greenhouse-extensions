# Greenhouse Extensions [![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

## About this project

Greenhouse Extensions is a collection of Kubernetes plugins and operators that extend the Greenhouse platform with Helm charts and operators, maintained by SAP SE and cobaltcore-dev contributors.

## Support, Feedback, Contributing

This project is open to feature requests/suggestions, bug reports etc. via [GitHub issues](https://github.com/cobaltcore-dev/greenhouse-extensions). Contribution and feedback are encouraged and always welcome. For more information about how to contribute, the project structure, as well as additional contribution information, see our [Contribution Guidelines](CONTRIBUTING.md).

## Security / Disclosure
If you find any bug that may be a security problem, please follow our instructions at [in our security policy](https://github.com/cobaltcore-dev/greenhouse-extensions/security/policy) on how to report it. Please do not create GitHub issues for security-related doubts or problems.
